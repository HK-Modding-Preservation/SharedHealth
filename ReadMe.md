# SharedHealth

A HKMP addon that allows synchronization of healing and damage across teams.

# Setup
When the mod and its dependencies are installed, all players need to be in the same team. Players not in a team will
not have their health synced. The sharing works per team, so you can have multiple shared healthbars on the same server.

# Known Issues
When taking damage with an inventory open (map, dialogue, charms, etc.) all kinds of unintended behavior can sometimes
happen (inputs stop working until another damage is taken, dialogue UI stays on screen, etc.). There is no way (that I know of) to fix this.
Let me know if you have any ideas on how to fix it.
